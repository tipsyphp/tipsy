tipsy

Github Examples Docs

Tipsy is an MVW (Model, View, Whatever) PHP micro framework inspired by AngularJS. It provides a very lightweight, easy to use interface for websites, rest apis, and dependency injection. It is also incredibly fast.


Example Usage


> index.php
$app->home(function($View) {
    $View->display('index', [user => 'crystal']);
});



> index.phtml
<h1>Hello <?=$user?></h1>




Installation

To install using composer use the command below. For alternate installation methods see Installation.

composer require tipsyphp/tipsy
